====================================================
  CEK APAKAH GAMBAR TERDETEKSI
====================================================

  Buka browser, akses:
  http://localhost/asetilmu/assets/images/books/[nama-file].jpg

  Contoh:
  http://localhost/asetilmu/assets/images/books/behind-the-smile.jpg

  ✅ Gambar muncul  = nama file sudah benar
  ❌ Error 404      = nama file salah atau file di folder yang salah

====================================================
  JIKA FOTO MASIH TIDAK MUNCUL
====================================================

  Masalah umum:

  □ Ekstensi tersembunyi di Windows
    → Di File Explorer: View → Show → File name extensions
    → Pastikan tidak ada ekstensi ganda: foto.jpg.jpg

  □ Browser cache
    → Tekan Ctrl + Shift + R untuk hard refresh

  □ File ada di folder yang salah
    → Harus di:  assets/images/books/  (folder ini)
    → Bukan di:  uploads/books/

  □ Ada spasi di awal/akhir nama file
    → Rename ulang, pastikan bersih
